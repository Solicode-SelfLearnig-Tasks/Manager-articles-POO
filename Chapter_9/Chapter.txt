Chapter 9: Encapsulation

In this chapter, I learned how to protect class data using Encapsulation.

Key things to remember:

1. Private Properties:
- Use `private` for properties so they can't be modified from outside the class.

2. Getters & Setters:
- Getters (`getVariable()`) are used to read the data.
- Setters (`setVariable($value)`) are used to update the data securely.

3. Validation:
- Always use Setters to check data before it's saved.
- Example: check length with `strlen()`.
- Example: clean bad inputs using `htmlspecialchars()`.

4. Clean Code:
- Give methods clear and readable names.
- Ensure each method focuses on doing exactly one thing.

Conclusion:
Encapsulation keeps our objects safe, reliable, and bug-free!
